Web Development Foundations: Web Technologies part II

Media and back-end technologies.

Front-End Technology

Common Image Types

  • Jpg, gif, png, svg - most commonly used web graphics
  • Common image concerns
  • JPEG - Joint Photographic Experts Group
  • GIF - Graphics Interchange Format
  • PNG - Portable Network Graphics
  • SVG - Scalable Vector Graphics

What is an API?

  • Application programing interface
  • Set of instructions and standards for accessing services or components of a specific system
  • Makes it easier for developers to use an application or to make services publicly available for anyone who wants to use them
  • Websites can leverage power of larger online applications to offer services and functionality far beyond their normal capabilities

HTML5 APIs

  • Plugins were created to get around HTML’s limitations
  • WHATWG created HTML5, then got supported by W3C
  • Media API
  • Drag and Drop API
  • App Cache API
  • Geolocation API
  • Canvas API
  • Designers can now build complex front-end interfaces with HTML, CSS, and JS that function more like applications than websites

Web Fonts

  • Arial, Verdana, Georgia were used back when we could only use widely used system fonts
  • Technology that allows for the downloading and temporary installation of fonts within the browser
  • Most frequently used to describe the CSS @font-face rule
  • How web fonts rule

Back-End Technology

Server-Side Scripting

  • Runs off web server
  • Professing forms, managing online purchases, registering user, booking hotels rooms
  • Used to be done through CGI (common gateway interface) to run C+ or PERL on websites
  • Now PHP, JAVA, .NET, CFML is used
  • Scripts run on server before content is seen by end user

The Importance of PHP

  • Had greatest impact
  • Free and open source and attracted wide community of developers
  • Paired with open source tools Linux, Apache and mySQL, became backbone of millions of sites across the world
  • Wordpress is built in PHP
  • Embedded within page’s HTML

Other Popular Server Languages

  • JSP - Java Server Pages
  • .NET: Visual Basic .NET
  • ColdFusion
  • Python
  • Ruby

Dealing With Data

  • Can be stored locally on client, in simply structured text files or within a database
  • Depends on type of data that needs to be saved and retrieval and usage
  • Cookies are written by browsers that can store small items like user prefs or session IDs
  • HTML5 introduced web storage API
  • XML - extensible markup language

Using SQL Databases

  • Structured Query Language
  • Standard language for managing and retrieving information from databases
  • SQL queries can be used to parse and use data within application
  • SQL syntax is simple and logical
  • There are multiple versions of SQL, each with its own extensions

Content Management Systems

  • Application that helps control creation, management, publishing and archiving of content
  • Can create groups of users
  • Some CMS are hosted online and only require an account to start building hosted sites
  • Other CMS can be installed on your own web server
  • Joomla, Drupal, Movable Type, ModX, Radiant, WordPress, ExpressionEngine

Content Delivery Networks

  • CDN
  • System for delivery content over a distributed network of servers
  • Network of servers all over the world holding same content, and can make delivery of content faster by using closest server
  • Largely used to serve static resources such as CSS and JS libraries and media files
  • Using Google’s Hosted Libraries reduces load from your server, speeds the delivery of jQuery to the client, and increases the likelihood that the client will have jQuery cached

Cloud Services

  • Distributed network of computers that can perform tasks by taking advantage of the scale and power of the network
  • Usually refers to distributed processes over the internet
  • Services usually accessed through apps
  • Ruby, .NET, Java and JS used to develop
  • Need large and complex server structures and need to be scalable

What is GitHub?

  • Online distribution service that allows users to store repositories online that can be version controlled and shared by users
  • Built around Git, an open source version control system by creator of Linux
  • Allows you to manage projects, publish files and store revisions
  • Offers both free and commercial plans
  • Git is command line tool

JavaScript Libraries

  • Collection of pre-written JS code that makes it easier to perform targeted tasks
  • JS libraries extend the functionality of native JS by adding methods and values for commonly performed tasks
  • Some libraries are extremely focused like Date.js
  • Others like jQuery have much broader focus
  • To use, need to include link to it in HTML
