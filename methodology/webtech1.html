Web Development Foundations: Web Technologies part I

Introduction to web concepts and front-end technologies.

Introduction to the Web

Understanding How the Web Works

  • Internet =/= web
  • Internet: massive network of million computers all over the world using protocols and languages
  • World Wide Web: means of transmitting data over the internet using HTTP and HTML
  • The web is series of interactions between clients and servers

Working with Clients and Servers

  • Static sites
    • Every time you type URL or click a link in your browser, a request is made to that server
    • Browser makes request with Uniform Resource Locator to Domain Name Server which translates URL into IP address. Browser then uses IP address to locate host server and send request for content.
    • If request is simple HTML page, it’s sent to browser. If there are images or CSS, those requests are presented to server and resources are downloaded and content is rendered as well. Also JS.
  • Dynamic sites
    • Must first be processed by web application such as PHP, .NET or Ruby before content is served by browser
    • Hotel reservation, real estate, etc.

The DNS and Why It Matters

  • Web sites have domain names that are associated with site’s IP
  • “Phonebook of the internet”
  • When you register a newly hosted website, you are given an IP address to go with your domain name

Internet Protocols

  • Different hardware, software and OS but need to all communicate with each other
  • Protocols are simple languages that let one computer share information with each other
  • Internet Protocol Suite
  • TCP/IP
  • Application, link, transport, internet layer

Exploring HTTP

  • Hypertext Transfer Protocol
  • Standard protocol for transferring data over the WWW
  • Browser is HTTP client while web server is HTTP server

Front-End Technology

Front-End Design

  • The visual (UI) layer of websites and applications
  • UI, layout, typography, images and other visual elements and their styling
  • Can often refer to usability and UX design as well

HTML, the Language of the Web

  • Essential skill for any and all web designers
  • Has a long history
  • HTML5 is standard today

Structuring HTML

  • Easy to learn

Controlling Presentation Through CSS

  • Cascading style sheets
  • Collection of formatting rules contained in external file
  • Cascade: how styles are applied to pages

Client-side Scripting with JavaScript

  • JavaScript was developed in 1995 by Netscape to increase functionality of its native browser and has evolved into standard scripting language of the web
  • Nothing to do with Java
  • Allows designers to create interactive elements
  • Can open windows or give users feedback
  • Can update, sort, and present data directly in the browser
