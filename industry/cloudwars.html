Cloud Wars

Cloud Wars: Amazon (AWS) vs. Google (GCP) vs. Microsoft (Azure)
Outline
  • Azure has been catching up to AWS
  • Cloud computing vs. traditional design
    • Traditional:
      • Massive up-front cost, big “steps”
      • IT staff: focus on the data center
      • In-house knowledge limits
      • Recreate the wheel
      • It’s yours
      • “Monster server” capabilities”
    • Cloud:
      • Pay as you go, pricing models
      • Elastic computing, grow as needed
      • Economy of scale
      • Immediate security accreditation
      • Multiple data centers easily
      • Collaborative innovation
      • Horizontal scaling
        • Biggest difference between the two
        • When you set up a data center, it costs a lot for the routers, switches, servers, etc; overprovisioning is a problem
        • In cloud, there’s little waste if any
        • Traffic patterns go up AND down (ex. holidays)
  • Building a cloud presence: Amazon AWS
    • Pick your region
      • What region of the world do you want to be in
    • Pick your availability zone
      • What zone inside region do you want to be in (data center)
    • Provision your server
    • Configure services
    • Expand to other availability zones
    • Expand to other regions
  • Understanding AWS Services
    • Cloudwatch
    • SimpleDB/DynamoDB
    • Elastic Computer Cloud (EC2)
      • Virtual server in the cloud
    • Simple Storage Service (S3)
      • Designed for massive amount of data storage
      • Can be web accessible
    • Elastic Block Store (EBS)
      • Storage your server uses
    • Route 33
    • Virtual PRivate Cloud (VPC)
    • Auto Scaling
      • Goes with EC2
      • Add or decrease servers as needed
      • Directly impacts cost
    • CloudFormation
    • Identity and Access Management (IAM)
    • Elastic Load Balancing (ELB)
  • Microsoft Azure
    • A lot of services mirror each other
    • Microsoft better for Enterprises (according to the guy who works for Microsoft lol)
    • Cloud products integrate with existing install base, making it easier to migrate to the cloud
    • “Premier ideal hybrid solution”
    • Visual Studio integrated into Azure
    • Azure Active Directory can autolink with Active Directory
  • Google’s Infrastructure
    • Been building their own private cloud for over 15 years
    • Google builds literally all their infrastructure
    • Fastest global network
    • Google build their public cloud on top of that infrastructure, enabling the world to build at the speed and scale of Google
    • Google made the big data scene
  • AWS scores highest in 4 out of 4 use cases
  • The best providers have an extensive array of additional services that extend across the spectrum from IaaS to platform as a service (PaaS).
  • Cloud industry is in a state of upheaval
  • Get certified in a cloud
  • Prices climb quick
