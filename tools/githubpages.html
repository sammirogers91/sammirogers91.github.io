My GitHub Page: sammirogers91.github.io


I had no problems setting up my Page. I'm already using GitHub and have been for about a month now, and I'll admit at first I was pretty confused about what GitHub actually was. It took me a lot of playing around with to get the hang of version control. Not having really played with HTML for many years, my mind was still stuck on FTP, so it was a bit of an adjustment to my prior way of thinking.


When I first copied and pasted my repo name into the address bar, it first took me to a page that simply displayed my README.md file, and I spent a good minute wondering what I had done wrong. It only took a simple refresh to get my index.html file displayed as it should. I guess there was a bit of a lag in creating the file. Don't be impatient, and when in doubt, refresh. :)
