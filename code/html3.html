HTML Essential Training: Part III

Styling and lists.

Creating Lists

Unordered Lists

  • Grouping content
  • Can nest them

Ordered Lists

  • Numbers instead of bullets
  • Can start at different number used start=”#”
  • Can reverse list using boolean attribute reversed

Definition Lists

  • Types that consist of two parts for each item: term and description
  • Also called description lists
  • Order most definitely matters
  • Terms usually rendered as top-level items and definitions are inset
  • Glossaries, FAQs, dictionaries, etc.

Controlling Styling

HTML and CSS

  • CSS is presentational syntax
  • Selectors, properties, values

Creating Inline Styles

  • Inline styles are usually applied to a single element and included in HTML code
  • Colon assigns value, end with a semicolon
  • Frowned upon because it’s not very efficient at all
  • Used for emails and scripting

The Style Element

  • Element selector {}
  • Much more flexible
  • Selectors change all elements referenced
  • .alert is a class selector that finds all elements with class attributes
  • H2.alert says to find element with specific class

Controlling Typography

  • Hexadecimal notation breaks down rgb between values of 0 and 255
  • Fallback fonts go after main font in font-family
  • Font-size em is relative
  • When margins are set, it goes clockwise
  • Whenever there’s a conflict, the more specific rule wins
